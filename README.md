# CV Analyzer

Aplicación para analizar currículums en PDF y evaluar candidatos según una oferta de trabajo usando IA.

## Características

* Subida de CV en PDF
* Análisis automático del contenido
* Comparación con una descripción de puesto
* Resultado con porcentaje de ajuste
* Identificación de habilidades, fortalezas y debilidades

## Instalación

```bash
git clone <repo>
cd cv_analyzer
pip install -r requirements.txt
```

## Uso

```bash
streamlit run app.py
```

Abrir en el navegador:
http://localhost:8501

## Cómo funciona

1. Subes un CV
2. Introduces la oferta de trabajo
3. El sistema analiza y devuelve una evaluación

## Requisitos

* Python 3.9+
* CV en PDF (con texto seleccionable)
